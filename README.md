## Healthcare Analysis Capstone Project
# Project Overview
This project focuses on analyzing healthcare data using Microsoft Excel to generate meaningful insights and support data-driven decision-making. The objective is to clean, analyze, and visualize healthcare-related data such as patient details, hospital performance, billing information, and treatment trends.
